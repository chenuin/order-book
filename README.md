# Order Book

## 環境準備
- 安裝 [Node.js](https://nodejs.org/)，建議版本：22.x 或以上
- 安裝 [pnpm](https://pnpm.io/) 套件管理工具

若尚未安裝 pnpm，可使用以下指令透過 npm 安裝：
```sh
npm install -g pnpm
```

## 安裝
Install packages
```sh
pnpm install
```

## 開發
Start dev server
```sh
pnpm dev
```

## 部署
Build for production
```sh
pnpm build
```

## Eslint
Analyzes the code base on customizable rules
```
pnpm lint
```
